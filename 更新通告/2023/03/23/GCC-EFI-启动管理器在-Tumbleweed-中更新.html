GCC、EFI 启动管理器在 Tumbleweed 中更新

2023-03-23 | Hanjingxue Boling | CC-BY-SA-3.0

openSUSE Tumbleweed 在本周发布的快照中包含了大量安全补丁、错误修复和新功能。

openSUSE 使用了新的编译器（GCC 13），用户使用 zypper dup 更新系统时会收到一次全量更新。

快照 20230319 是一个全量重构快照，提供了一个 GCC 13.0.1+git 更新；该更新 rebase 了一个补丁并启用了一个互斥锁（mutex）链接。flatpak 更新至 1.14.4，更新了翻译并解决了两个 CVE：CVE-2023-28101 和特定于虚拟控制台的 CVE-2023-28100；建议用户使用 GNOME Software 等图形用户界面，而不是 xterm、gnome-terminal 或 Konsole 等图形化终端模拟器。用于单指令、多数据的 C++ 库，highway 1.0.4 的更新提供了更快的 KV128 排序。该软件包还更新了 1.0 草案的 RISC-V Vector Extension Intrinsics。其他要更新的包是 libstorage-ng 4.5.86 以及几个 libqt5 包。

快照 20230318 仅更新了两个包。fcitx5-gtk 更新到 5.0.22 版本。这个 gtk-im-module 和基于 glib 的 dbus 客户端库实现了 notify-focus-out 信号并更改 GtkIMContext.reset 以始终提交预编辑状态。另一个要更新的包是用于创建 MusicBrainz DiscID 的库：libdiscid；这是一个很棒的开放音乐百科全书，可以收集音乐元数据并将其提供给公众。libdiscid 0.6.4 修复了编译器错误并要求 CMake 2.8.12 作为最低版本。

快照 20230317 将 DNS 协议 bind 更新至 9.18.13。该更新提供了多项新功能，例如提高在成功传输 RPZ 区域后应用的命名响应策略区域 (RPZ) 更新的响应能力。Plasma 5.27.3 更新中发布的错误修复可以让 KDE 爱好者感到满意。一些突出的修复包括在映射中添加表情符号选择器、从历史记录加载时删除重复项以及 PowerDevil 试图进行一些更改以避免浪费宝贵的电量。gtk4 4.10.1 的更新带来了大量的变化。除了删除上游已修复的补丁外，新版本还修复了内存泄漏、一些滚动问题并改进了跨平台小部件工具包的搜索性能。systemd 更新到 253.1 版本，添加了一些补丁，其中一个直到开发者修复 dracut 中的 LVM 引导故障前，是一种临时解决方法。快照更新了其他几个包，包括 pipewire 0.3.67、icewm 3.3.2 和许多 qt6 包。

快照 20230316 共有三个主要版本更新，其中之一是可扩展固件接口（EFI）启动管理器 efibootmgr。efibootmgr 18 恢复了激活错误消息并修复了帮助消息。该软件包还添加了一个用于插入新条目位置的选项，并修复了简单的运行示例。第二个主要版本更新是针对 efivar 38 更新中的 EFE 变量。此软件包修复了 nvme-subsystem 设备的解析，添加了一些新工具并正确检查 mmap 返回错误。第三个主要版本更新是 BitTorrent 客户端 transmission 4.0.2。新版本解决了一些潜在的崩溃问题并修复了 IPv6 跟踪器 URL 的显示。重构的 Web 客户端现在支持移动端设备使用。Linux Kernel 是此快照中唯一不是主要版本更新的更新。kernel-source 6.2.6 恢复了一部分 wifi 配置并删除了一些 Realtek 无线驱动程序。

原文：GCC, EFI Boot Manager Update in Tumbleweed，作者：Douglas DeMaio
