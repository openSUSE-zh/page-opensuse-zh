Audacity、OpenSSL 和 systemd 在 Tumbleweed 中更新

2023-02-09 | Hanjingxue Boling | CC-BY-SA-3.0

openSUSE Tumbleweed 在过去的一周发布了一些快照；并为二月份的第一个快照启动了自动迁移。

本周的快照所涵盖的软件包包括但不限于 GCC、GStreamer、KDE Gear、博客标题中提到的数个软件包。

快照 20230208 在本周为 ImageMagick 提供了第二次更新，ImageMagick 7.1.0.61 版本改进了一些文档，并移动了 -set profile 命令行界面处理。fetchmail 6.4.36 对多国语言翻译进行了更新。xwayland 22.1.8 的更新修复了第二种可能的带外远程访问 OOB 访问。向后兼容的 xwayland 还修复了 CVE-2023-0494，它可能允许在 X 服务器运行特权和远程代码执行 SSH X 转发会话的系统上进行本地提权。快照还有其他几个软件包更新，包括 hwdata 0.367、ncurses 6.4.20230128、texinfo 7.0.2、ceph 等。

快照 20230207 对网络基础设施包 dnsmasq 2.89 进行了更新。dnsmasq 修复了导致 DNS 缓存内部数据结构损坏和“缓存内部错误”日志记录的错误。变更日志指出，“该情况只在一个地方出现过，甚至生成一个测试用例来复现错误都花费了我们相当大的努力，但无法确保它不会发生，其效果是严重破坏了缓存。” SELinux 的策略分析工具 setools 更新到 4.4.1 版本并更新了权限映射。setools 还进行了一些代码清理，并替换了一个已在 NetworkX 3.0 版本中被删除的弃用函数。webkit2gtk3 2.38.4 更新改进了编解码器选择逻辑，更好地处理延迟，并改进了帧丢弃以避免音频与视频不同步。kernel-firmware 20230125 和 Linux kernel-source 6.1.10 的更新似乎分别有几个 AMD 添加和 arm64 修复。

快照 20230204 将音频软件 audacity 更新至 3.2.4 版本。由于此次升级，音频现在可以在 audio.com 上公开共享。Audacity 还添加了带有剪切、复制和粘贴按钮的新工具栏。openSUSE 在 KDE Gear 22.12.2 发布后迅速将更新包送入风滚草用户手中。文件管理器 Dolphin 修复了在子目录无法打开时目录的大小。更新后的 Kalendar 修复了启动崩溃问题。视频编辑器 Kdenlive 还修复了崩溃和未保存子剪辑的屏幕撕裂问题。libstorage-ng 4.5.68 更新了格鲁吉亚语翻译。php8 8.1.15 有多个修复，包括修复 opcode 更新后块优化传递中的错误比较。该软件包还改进了对速度优化的哈希算法 XXH3 的处理。

快照 20230202 将网络浏览器 Mozilla Firefox 更新至 109.0.1 版本；修复了一些表情符号字符。NetworkManager 1.40.12 的更新修复了在 IPv4 共享模式下并发调用 iptables 的问题。用于配置和自定义字体访问的库 fontconfig 已更新至 2.14.2 版本。fontconfig 修复了描述中的拼写错误，调整了缩进并添加了渲染选项。

快照 20230202 将 OpenSSL 从 1.1.1s 版本变更为 3.0.7 版本。新的 OpenSSL 版本被设置为默认版本，并且是一个跨越很长一段时间的重大项目，以使其可供用户使用。这些更改放宽了回归测试的加密策略要求，并删除了一些补丁。OpenSSL 3.0 是一个主要版本，有许多软件包必须得到调整。OpenSSL 3.0 有很多改进。构建和安装过程发生了重大变化，许多结构在新版本中变得不透明。迁移指南中提供了更多信息。

文本编辑器 vim 9.0.1270 有多个修复，包括一些缩进过多的代码和一个现在可以识别 NetworkManager 连接的修复。快照 20230202 更新了 GStreamer 1.22.0 及其几个相同版本的插件。快照进行了一些 AV1 视频编解码器改进和一些 WebRTC 支持工作。快照还提供了用于 Amazon AWS 存储和音频转录服务的新插件。

快照 20230201 更新了一些软件包，例如 gcc13。gcc13 13.0.1 plus 版本增加了对新前端 Rust 和 Modula-2 的支持。GCC 还修复了 Go 前端以修复 s390x 上的失败构建。本月的第一个快照具有重要意义，因为它启动了与 i586 从 factory 分离的 zypper dup 自动迁移。根据 openSUSE 上一次发布工程会议的记录，由于重建计数器在项目之间不同步，因此更改软件源可能会导致一系列软件包降级。到三月底，预计所有的用户都将完成迁移；到那时，除了在 factory 代码库的 Staging:O 工作流程中确定的大约 1,800 个包之外，Tumbleweed 将禁用新的 i586 软件包的构建或发布。

一月末的快照 20230131 为图像编辑器 ImageMagick 提供了第一次更新，更新至 7.1.0.60 版本。ImageMagick 只收到三个提交，主要是清理一些代码。systemd 252.5 更新引入了一个预设，允许 MicroOS 用户使用 systemd 用户单元，并添加了一个事务更新通知程序，允许（针对云和容器部署进行了优化的）发行版的用户在事务更新成功或失败时获得桌面通知。另一个在快照中更新的包是 xterm 378，它改进了一些描述和几个检查，比如改进了对不受支持的格式化字符的检查。

原文：Audacity, OpenSSL, systemd Update Tumbleweed，作者：Douglas DeMaio
