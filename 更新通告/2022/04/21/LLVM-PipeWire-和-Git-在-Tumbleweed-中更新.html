LLVM、PipeWire 和 Git 在 Tumbleweed 中更新

2022-04-21 | Hanjingxue Boling | CC-BY-SA-3.0

自上周四以来，openSUSE Tumbleweed 已经发布了三个快照。

如果快照 20220420 通过 openQA，可能会在本文发布之前发布，并将发布的快照数量增加到四个。

快照 20220419 中更新了不到 10 个包。快照中的大多数更新来自 Linux 内核 5.17.3 更新。此次更新针对 x86 进行了一些 KVM 修复；还有一个针对 arm64 的修复程序，可确保不更改事件过滤器。内核更新中还有大约 30 处针对 DRM 更改。使用 JACK 后端的 Wine 应用程序不应再因 pipewire 0.3.50 更新而崩溃。音频和视频包更新也有一个变化，确保 ALSA 现在只分配一个足够大的缓冲区大小来容纳四倍的定量限制，而不是尽可能大。libnl3 3.6.0 的更新添加了对 IPv6 的通用路由封装和虚拟隧道接口支持，并且 yast2-trans 和 libstorage-ng 4.5.4 更新了斯拉夫语翻译。

快照 20220415 将 ImageMagick 更新到 7.1.0.29。根据更新日志，此次更新进行了一些恢复，并修复了以 RGBA 形式导入的灰色图像。Mozilla Firefox 99 的第一个次要版本是 99.0.1。 火狐浏览器更新修复了下载面板中的选择与拖放的问题。此次更新还修复了阻止 Zoom 画廊模式工作的问题。git 2.35.3 更新修复了一些 CVE；根据变更日志，CVE-2022-24765 可能允许 git 从意外的工作树执行其他用户定义的命令。快照中要更新的其他软件包包括 vim 8.2.4745、Ruby 3.1.2、Xen 4.16.1 和 whois 5.5.13 等。

上周四的 20220414 快照将 procps 从 4.0.0 版本恢复到 3.3.17 版本。这个主要版本恢复了几个补丁并处理了一些 CVE。LLVM 14.0.0 的主要版本更新在快照中发布。这个版本引入了许多新工具，删除了一些补丁；受 GCC 打包方式的启发，并选择拆分 Clang 库。快照中对 libunistring 1.0 进行了重大更新，提供了 Unicode 14.0.0 支持和许可证更改。本次快照有一个 kdump 1.0.2 的 git 版本更新，它为 fadump 的读写提供了文件系统重新挂载。快照中还更新了 ncurses 和 dracut。快照中还更新了其他几个包。
