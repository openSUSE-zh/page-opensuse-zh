Tumbleweed 继续连续发布快照

2022-09-01 | Hanjingxue Boling | CC-BY-SA-3.0

Tumbleweed 连续每日发布快照的次数达到了惊人的 26 次。

openSUSE Tumbleweed 本周依旧快速滚动中，glibc、ibus、Mozilla Firefox 和 sudo 等软件包都收到了更新。

在快照 20220829 之后，这种情况还会继续吗？ 用户应该很快就会知道。

快照 20220829 为 AppArmor 和 libapparmor3.0.7 提供了更新。新版本修复了 buildpath.py 中的 setuptools-version 检测。日文用户手册 man-pages-ja 20220815 在本次快照中更新，进行了一些改进。tree 2.0.3 版本更新为 .gitignore 功能提供了多个修复，并修复了几个段错误。

快照 20220828 更新了十个包。其中包括 ibus 1.5.27，它在 GNOME 桌面中启用了 ibus 重新启动，并在 Plasma Wayland 中禁用了 XKB 引擎。webkit2gtk3 2.36.7 的更新修复了几个崩溃和渲染问题，并解决了一个与 Apple 使用该软件包遇到的相关 CVE。Python web 框架和异步网络库 python-tornado6 6.2 默认启用 SSL 证书验证和主机名检查，其持续集成已从 Travis 和 Appveyor 转移到 Github Actions。快照中要更新的另一个包是字体处理程序 libXfont2 2.0.6。libXfont2 新版本修复了一些拼写和措辞问题；它还修复了注释以反映删除了对旧版 OS/2 的支持。

Mozilla Firefox 浏览器新的大版本在快照 20220827 中发布。Firefox 104.0 解决了多个 CVE，包括地址栏欺骗以伪装 URL 的 CVE；另一个修复了一个漏洞，该漏洞可显示内存损坏的证据以及可被用于运行任意代码的可能性。GNU C 库的更新增加了重要的新特性；glibc 2.36 添加了 process_madvise 和 process_mrelease 函数。glibc 添加了对 DT_RELR 重定位格式的支持，并添加了套接字连接 fsopen 和许多其他排序功能。VMware 的 open-vm-tools 12.1.0 启用了多项功能，可以更好地管理用户与虚拟机的无缝交互，修复了允许本地权限提升的漏洞；它还修复了用于 32 位 Linux 版本的 ContainerInfo 插件的构建。快照中还更新了一些 RubyGems，例如 rubygem-faraday-net_http 3.0.0、rubygem-parser 3.1.2.1 和 rubygem-rubocop 1.35.1。

快照 20220826 总共更新了三个包。简单的 PIN 或密码短语安全阅读器 pinentry 更新到 1.2.1；pinentry 改进了可访问性并修复了初始化期间的错误处理。更新后的 pinentry 还确保了始终从内存中清除输入的 PIN。它的图形用户界面 pinentry-gui 也更新到了 1.2.1 版本。将 UNIX 密码文件转换为影子密码格式的 shadow 已更新至 4.12.3 版本。shadow 修复了一个有 9 年历史的 CVE。CVE-2013-4235 修复了在复制和删除目录树时 TOCTOU 的竞争条件存在的漏洞。该软件包还更新并修复了一些西班牙语和法语翻译。

sudo 1.9.11p3 的次要版本更新在快照 20220825 中发布。sudo 修复了 Python 模块在某些系统上与 Python 3.9.10 的崩溃，并且 AppArmor 集成可用于 Linux，因此 sudoers 规则现在可以指定一个 APPARMOR_PROFILE 选项来运行一个由命名的 AppArmor 配置文件限制的命令。sudo 还修复了 1.9.11p1 中引入的回归，该回归从未在 Tumbleweed 快照中发布；如果命令没有返回输出，则在登录到 sudo_logsrvd 时此漏洞会导致警告。开源磁盘加密包 cryptsetup 更新到 2.5.0 。这个新版本从项目中删除了 cryptsetup-reencrypt 工具，并将 reencryption 移动到已经存在的 cryptsetup reencrypt 命令。快照中要更新的其他软件包包括 gnome-bluetooth 42.3、设备内存启用程序 ndctl 74、yast2-tune 4.5.1 等。

原文：Tumbleweed Continues Release Streak，作者：Douglas DeMaio
