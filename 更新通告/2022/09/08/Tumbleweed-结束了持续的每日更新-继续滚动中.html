Tumbleweed 结束了持续的每日更新，继续滚动中

2022-09-08 | Hanjingxue Boling | CC-BY-SA-3.0

Tumbleweed 连续每日发布新快照的记录在上周以 26 个快照的新记录结束，但 openSUSE Tumbleweed 似乎并没有随着快照持续发布的频率而放缓。

快照 20220829 打破了连胜纪录，但 Tumbleweed 本周继续快速前进，并发布了几个快照和软件包更新。

在详细讨论这些快照之前，GNOME 43 可能需要一些援助才能将其纳入 Tumbleweed 快照中。如果你有兴趣，请阅读 @openSUSE 的推文并在 https://matrix.to/#/#gnome:opensuse.org 上与团队进行沟通。

The @gnome 43 release candidate is packaged up in @openSUSE's GNOME:Next, which is our unstable branch. The extensions haven't been tested. Looking for people to test. Chat at https://t.co/iEBQH4ZRt9 for more info. pic.twitter.com/nTJuOkZ6ae

— openSUSE Linux (@openSUSE) September 7, 2022

最新发布的快照是 20220907。gnome-bluetooth 42.4 的更新使其 API 接口现在可以导出 UPower 中列出的所有蓝牙设备的电池信息。microos-tools 2.16 更新了 MicroOS 的文件和脚本。libsoup 3.0.8 的更新对 HTTP/2 的可靠性进行了大量改进，并使用默认代理解析器修复了 http 代理身份验证。SVG 渲染库 librsvg 已更新至 2.55.1 版本，并正在尝试为 librsvg 提供奇偶版本控制；奇数次要版本将被视为不稳定，偶数次要版本将被视为稳定。这应该很有趣！快照中更新了其他一些软件包，包括 kernel-firmwarel 20220902 和 yast2-vpn 4.5.1 等。

Linux 内核在快照 20220906 中从 5.19.2 更新到 5.19.7。随着内核中的 amdgpu 驱动程序更新的同时，用户还收到一些 btrfs 修复和更改。用户空间工具集软件包 lvm2 2.03.16 的更新修复了一些段错误，添加了几个补丁并修复了精简池扩展上已删除消息的丢失。Plymouth 22.02.122 的更新对虚拟现实进行了更改，因为不需要在虚拟现实眼镜上显示启动屏幕。或者需要？文本编辑器 vim 9.0.0381 修复了一些崩溃、闪烁和发出警告的 Clang 静态分析器。快照中还对 sqlite 3.39.3 和其他软件包进行了更新。

快照 20220905 中仅更新了两个包。网络映射工具 nmap 发布 7.93 版本以纪念 Nmap 发布 25 周年。恭喜！该工具升级了几个库并确保 Nmap 使用 OpenSSL 3.0 构建，同时不使用已弃用的 API 函数。该软件包还修复了一个错误，该错误会阻止它在未配置 IPv4 地址时发现 Linux 上的接口。另一个要更新的软件包是 Thunderbolt 3 设备管理器 bolt 0.9.3。该更新围绕 glib 创建了一个工作。

快照 20220904 更新了十几个软件包。其中包括：AppStream 0.15.5、git 2.37.3、libstorage-ng 4.5.43、libzypp 17.31.1、zypper 1.14.56 和 gc 8.2.2。git 的更新纠正了一些回归并进行了一些 git fsck 改进。openSUSE 的 zypper 确保 up 尊重求解器相关的命令行界面选项。

快照 20220903 对 btrfs 进行了更新。文件系统 btrfs 的更新为 LZO 和 ZSTD 压缩提供了一些可选支持；该更新还带来了与 glibc 2.36 的一些兼容性。GCC 12.2.1+git416 的更新最近从主干向后移植了一些修复，并对 armv7l 架构进行了一些更改。像素编码器和色彩空间转换器包 babl 0.1.96 删除了一些补丁并修复了单指令多数据 (SIMD) 的非对齐数据崩溃。其他要更新的软件包是 libvirt 8.7.0、lsscsi 0.32、upower 1.90.0 和几个 RubyGems。

自连续更新中断以来发布的第一个快照是快照 20220901。此快照更新了 curl 7.85.0。新版本中有 79 位贡献者；此次更新还修复了一个 CVE。CVE-2022-35252 是在 curl 4.9 中引入的，允许“姐妹站点”拒绝为兄弟站点提供服务。该版本更新还增加了对传输层安全协议 1.3 的支持。NetworkManager 1.40.0 在本次快照中更新；如果设备上的 MAC 发生变化，现在可以重新启动 DHCP。NetworkManager 也进行了一些国际化和本地化更改。Flatpak 1.14.0 的更新改进了对在两个本地设备之间传输文件的支持，称为侧载。该软件部署和包管理实用程序还为 libcurl 添加了包配置，以启用新的 HTTP 后端。快照中要更新的其他软件包包括 ncurses 6.3.20220820、grilo 0.3.15、mariadb 10.8.3、一个 kdump git 更新等。
