Git、curl 和 systemd 在 Tumbleweed 中更新

26. Nov 2021 | Hanjingxue Boling | CC-BY-SA-3.0

openSUSE Tumbleweed 过去一周每天都向滚动发布用户提供快照。

最新发布的快照是 20211124。这个快照带来了 systemd 249.7，它专注于包测试和测试套件的更新依赖项。文本编辑器 vim 8.2.3640 版本是一个小更新，但它充满了许多修复； 一些修复包括处理与 GTK 相关的内存泄漏、崩溃和性能问题。本次快照在 dracut 的更新中删除了一个多余的脚本头，并为 Linux-boot 进程包中的 zstd 设置了最佳压缩参数。快照中要更新的其他软件包包括 autoyast2 4.4.22、嵌入式 Linux 库 ell 0.46、GNOME 的文档查看器 evince 41.3 和 gtk-vnc 1.3.0。

快照 20211123 更新了 git 2.34.0，去掉了git rebase 的 --preserve-merges 选项。该软件包还调整了 git add、git mv 和 git rm 以避免更新 sparse-checkout 定义之外的路径，除非用户指定 --sparse 选项。C 客户端库 mariadb-connector-c 3.2.5 删除了一个回调函数，因为它无法被清除；受影响的版本等于或小于 OpenSSL 1.0.2。 快照中还更新了 yast2-packager 4.4.14、yast2-schema 4.4.6 和 yast2-trans； 后者更新了日语、斯洛伐克语和加泰罗尼亚语的翻译。

本周的第二个 Linux 内核更新在快照 20211122 中发布。通过 kernel-source 5.15.3 更新，修复了 Clevo 计算机的高级 Linux 声音体系。KVM 进行了一些更改，其中包括一项用于处理动态 MSR 寄存器（Model Specific Register）拦截切换的更改。Daniel Stenberg 提供了有关 curl 7.80.0 更新的视频内容；该版本是第 204 个版本，添加了 SHA256 指纹支持并修复了如果无法将 SSL 会话添加到缓存中的内存泄漏。xfsprogs 更新至 5.14.0 ，并引入了 liburcu 支持。yast2-bootloader 4.4.8 包配置了自定义引导分区，使其更加健壮和直观。其他要更改的 openSUSE 软件包是 snapper，它从版本 0.9.1 降级到 0.9.0，以及 transactional-update 3.6.2，它修复了如果挂载点设置了 unbindable 挂载标志而无法运行的几个应用程序。Web 框架包 python-falcon 的主要版本有一个升级到 3.0.1 的小版本更新；此次更新包括基于 ASGI 的 asyncio 和 WebSocket 支持；该软件包通过增强现有功能带来了更好的错误处理。 在这次和前一天的快照中，有几个 pypi 软件包被更新。

在快照 20211121 中更新的 pypi 包是 python-requests 2.26.0、python-importlib-resources 5.4.0 和 python-packaging 21.2，它们放弃了对 Python 2.7、Python 3.4 和 Python 3.5 的支持。 随着 systemd-rpm-macros 14 的更新，上游提交的向后移植变得可用。快照中要更新的另外两个包是 util-linux 2.37.2 和 util-linux-systemd 2.37.2。

systemd 249.6 在快照 20211120 中发布，GitHub 上有完整的更改列表；更新包中也有大量文档改进。firewalld 1.0.2 的更新修复了一些与 nftables 相关的拼写错误和分类。来自 YaST Sprint 的 yast2-storage-ng 4.4.14 的 10 月份更新调整了 YaST 在 fstab 文件中引用 LUKS 设备的方式，以使 systemd 更容易处理某些情况。 快照中更新的其他软件包是 php7 7.4.26、libstorage-ng 4.4.57 和 ncurses 6.3 等。

Mesa 21.3.0 是快照 20211119 中要更新的软件包之一。Meda 更新使 Panfrost 驱动程序正式符合 GLES 3.1。nodejs16 16.13.0 的更新带来了一个实验性的 ESM Loader Hooks 应用程序接口。ImageMagick 7.1.0.14 的更新修复了解析恶意 ps 图像文件时堆栈溢出的问题。该图像编辑器将不再将 PSD 文件的配置文件复制到所有图像； 只有第一个返回的图像将包含配置文件。 要重新启用旧行为，请添加 -define psd:replicate-profile=true。snapper 更新至 0.9.1 版，旨在修复一些 systemd 沙箱，然后在稍后的快照中进行回溯。 许多 YaST 包在快照中更新，如 yast2 4.4.22、autoyast2 4.4.21、yast2-installation 4.4.22、yast2-packager 4.4.13 和 yast2-update 4.4.4。YaST 博客中定期描述了这些包的更新。

从本周开始，快照 20211118 给了用户一个微笑的理由 —— GNOME 的网络摄像头应用程序 cheese 41.1的更新，该更新更新了项目描述和翻译。iproute2 5.15 的更新添加了链路聚合控制协议主动支持，python-argon2-cffi 21.1.0 被修改为跳过 Python 2 的构建，因为它不再受支持。Linux 内核在 Tumbleweed 中从 5.14.14 更新到了 5.15.2 ，开始了连续一周的快乐的快照。祝你们玩得很开心！
