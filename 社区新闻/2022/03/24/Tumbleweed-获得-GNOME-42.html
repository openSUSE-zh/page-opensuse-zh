Tumbleweed 获得 gnome 42

2022-03-24 | Hanjingxue Boling | CC-BY-SA-3.0

openSUSE 滚动分支 Tumbleweed 快速地为用户提供了最新发布的 GNOME 42。

这个由 gnome 贡献者们发布，备受关注的版本已在快照 20220323 中发布。

GNOME 42 具有新的全局深色用户界面样式偏好，并配备了重新设计的屏幕截图功能。GNOME 42 核心应用程序已移植到 GTK4 并提供下一代功能。GNOME 42 包括一组有价值的性能改进，例如视频现在使用具有硬件加速解码的现代 OpenGL 小部件；输入处理得到了显着增强，从而降低了输入延迟并提高了响应能力。全屏应用程序呈现方式的改进降低了视频播放的能耗并提高了游戏的帧速率。VLC 3.0.17.3 在快照中进行了更新，并修复了可能导致自适应流播放缺少音频的回归。快照中更新了其他几个软件包，包括 NetworkManager 1.36.4、GTK4 4.6.2、webkit2gtk3 2.36.0、yast2-installation 4.4.50 等。

于前一天发布的快照 20220322 提供了三个包更新。以太网和 IP 地址配对包 arpwatch 3.2 添加了一个配置选项来指定发送邮件的路径。aws-cli 1.22.65 更新了规范文件并进行了一些 API 更改，例如添加用于自定义插件删除 DeleteCustomPlugin 的操作以及添加 Hybridcast 作为 Dash Origin Endpoints 的可用配置文件选项。python-pip 更新至 22.0.4 版本；该包与 Python 3.6 不兼容，所以不适用于 openSUSE Leap 15。但是，此次更新修复了 pip 不考虑依赖关系的问题。之前的 20.2.4 版本在滚动版本中有大量更新。

快照 20220321 中更新了另外三个包。此快照为 GCC 11 提供了一个小更新，其中包含一个补丁来修复 i586 处理器上游戏 0 A.D. 的嵌入式预编译错误。C/C++ 代码分析工具 cppcheck 更新到 2.7.1 版本。该工具对检查器进行了各种改进，并增加了对容器视图的支持；视图属性已被添加到容器库标签中，以指定该类是一个视图。寿命分析已被更新，以使用这个新属性来查找悬空的寿命容器。另一个要更新的包是 rpm-config-SUSE，它改变了版本结构，从 0.g96 更新到了 20220317。

openSUSE 在本周末继续发布快照，其中快照 20220320 更新了 15 个软件包。此快照将 gnome-software 更新到版本 41.5，以添加几个与 appstream 相关的修复；此更新还禁用了特色轮播上的鼠标滚轮。本次快照将 container-selinux 从 2.171.0 更新到版本 2.180.0，并提供了几个附加选项，并允许用户角色使用容器域。libstorage-ng 4.4.94 的更新不再阻止扩展分区上的 blkdiscard；whois 客户端 5.5.12 更新了 .pro 顶级域服务器。其他值得注意的更新包是 librsvg 2.52.8、python-kiwi 9.24.29、yast2-installation 4.4.49 等。

快照 20220319 更新了 NetworkManager 1.36.2，现在可以加载插件目录中找到的所有已知插件。NetworkManager 还将 iproute2 和 iputils 从需求依赖转换为推荐依赖。Flatpak 1.12.7 在快照中进行了更新，并有一个变化，允许用 Flatpak 1.13.x 构建的应用程序在 share/metainfo 中导出 AppStream 元数据，这可能会在即将举行的 Linux 应用程序峰会上讨论。Linux 内核 5.16.15 似乎提供了一些 ARM 修复，并列出了 CVE-2022-0886 的修复，这是 CVE-2022-27666 的复制品。

于上周五发布，开启本周的快照是 20220318。该快照将 rubygem-formatador 从 1.1.0 恢复到 0.3.0。gstreamer-plugins-rs 0.8.2 版本中有一个 cargo.lock 的更新；desktop-translations 84.87.20220316 的第六个月更新 9301f89b 完成了几百个翻译变更。快照中的其他更新包括 fwupd-efi 1.2、p11-kit 0.24.1 和 yast2-packager 4.4.26。
